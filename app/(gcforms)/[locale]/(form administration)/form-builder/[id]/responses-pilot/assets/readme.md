## What you’ll get depending on your selection:

- **CSV (spreadsheet)** file that can have data appended to it as you get more responses. _Please do not rename this file._
- **Individual HTML copies** of responses in a folder for processing submissions one at a time. 
- **Files attached** in a folder separate from answers, if your form includes a file upload component.

## You’ll also see: 

- A folder with **raw data** in JSON format.  _These files eliminate the need for a manual confirmation step._ 
- A folder with **logs** of all actions taken. _These files help with debugging. Please do not alter these files._

## Ce que vous obtiendrez en fonction de votre sélection :

- Un fichier **CSV (feuille de calcul)** auquel nous pourrons ajouter des données à mesure que vous recevez davantage de réponses. _Veuillez ne pas renommer ce fichier._ 
- Un dossier avec les **copies HTML individuelles** des réponses pour traiter les soumissions une par une. 
- Un dossier distinct contenant les **fichiers joints** séparés des réponses, si votre formulaire comprend un composant de téléverseur de fichiers.

## Vous verrez également :

- Un dossier avec les **données brutes** au format JSON.  _Ces fichiers éliminent la nécessité de l’étape de confirmation manuelle._
- Un dossier contenant les **journaux** avec toutes les actions effectuées.  _Ces fichiers facilitent le débogage. Veuillez ne pas modifier ces fichiers._
