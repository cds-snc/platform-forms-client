GC Forms Schema - Alpha

Type: object Default: {}

A schema to define a Forms configuration.
Example:

{
    "internalTitleEn": "",
    "internalTitleFr": "",
    "publishingStatus": false,
    "securityAttribute": "",
    "submission": {
        "email": ""
    },
    "form": {
        "version": 1,
        "titleEn": "",
        "titleFr": "",
        "emailSubjectEn": "",
        "emailSubjectFr": "",
        "layout": [],
        "brand": {},
        "elements": [],
        "startPage": {},
        "endPage": {}
    }
}

Type: boolean

Used only for checkboxes, ignored on other elements. When set to true, this will require that all checkboxes be selected. Used typically for the attestation section.
